Google Chart Tools AngularJS Directive Example

This is probably the most simple example you can get. It just shows you what you need to get the Google Chart Tools AngularJS Directive to work.

Here are the steps:

  1. Download ng-google-chart.js from github and add a script tag to your html.
  2. Create a div like: 
     <div google-chart chart="chart" style="{{chart.cssStyle}}"/>
  3. Add 'googlechart' to your module like this: 
    angular.module('myApp',[ 'googlechart', ...
  4. Populate the $scope.chart like this:
{{chart | json}}
